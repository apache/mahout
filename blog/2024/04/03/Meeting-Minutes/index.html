Mahout Team
Apache Mahout

Weekly community meeting​

Subscribe to the Mahout User list to ask for details on joining.

Attendees​

  • Tommy Naugle
  • Andrew Musselman
  • Trevor Grant

Old Business​

  1. Happy hour some week soon, invite collaborators and conspirators
  2. Kernel method research spike: https://issues.apache.org/jira/browse/MAHOUT-2200
  3. Make ticket to add notebooks to notebooks directory in source tree (https://issues.apache.org/jira/browse/MAHOUT-2198)
  4. Add execute method to qumat https://issues.apache.org/jira/browse/MAHOUT-2201
  5. Splitting Mahout project code into discrete repos https://issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2204
    • mahout-website
    • mahout-classic
    • mahout-samsara
    • mahout-qumat
  6. Create testing suite template (PR open) https://issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2202
  7. Docker image for web site build https://issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2165
  8. Cirq backend (resolved) issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2203
  9. Moving to GitHub Issues instead of JIRA issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2205
  10. Next steps for qiskit and cirq back ends issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2206

New Business​

  1. Qumat allowed python versions incompatible with latest cirq requirements (PR open) issues.apache.org/jira/projects/MAHOUT/issues/MAHOUT-2207
  2. Reviewed pytest pr
  3. Reviewing happy hour date/time (done)
  4. Roadmap
    • Q2
      1. Qumat with hardened (tests, docs, CI/CD) cirq and qiskit backends
      2. Kernel methods started
      3. Submit public talk about Qumat
      4. Classic in maintenance mode
    • Q3 and beyond
      1. Amazon Braket
      2. Distributed quantum solvers
